## Building

```bash
# Configure
cmake -B build -DCMAKE_BUILD_TYPE=Release

# Build
cmake --build build

# Run
./build/TaskScheduler
```

## Neovim CMake Commands

- `<leader>cg` - Generate CMake configuration
- `<leader>cb` - Build project
- `<leader>cr` - Run project
- `<leader>cc` - Clean build
- `<leader>ct` - Select build target
- `<leader>cy` - Select build type (Debug/Release)
